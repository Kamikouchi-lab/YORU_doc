About these metrics

Stars / Forks / Contributors
Current counts from the YORU GitHub repository.
Total Downloads
Sum of all GitHub Release asset download counts across every version.
Clones (30d)
Number of git clone operations in the last 30 days. Based on full clones tracked by GitHub.
Page Views (30d)
GitHub repository page views in the last 30 days.
Unique Cloners / Visitors (30d)
Distinct users (by IP) who cloned or visited the repository within the last 30 days.
Top Referrers
Websites that directed the most traffic to the repository (last 14 days, GitHub Traffic API).
Popular Content
Most visited pages within the repository (last 14 days, GitHub Traffic API).
Last Updated
UTC timestamp of the most recent automated data collection via GitHub Actions.

Traffic metrics are collected daily by a GitHub Actions workflow and stored in this repository. Some metrics depend on collection timing and may not cover days when the workflow did not run.
